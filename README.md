## 项目简介

本工程为电赛调试专用平台，主要功能包括：

- 通过 **INA226** 实时采集系统电流、电压与功率，并记录最大功率
- 通过 **LCD4002（I²C 接口）** 将测量数据实时显示在 40×2 字符液晶屏上
- 通过 **4×4 矩阵键盘** 选择工作模式与参数编号
- 通过 **UART** 向外部设备发送控制指令，并接收距离/长度测量结果
- 支持 **线性 CCD**、**电机驱动**、**编码器**、**IMU**、**无线通信** 等多种外设的扩展接入

## 功能特性

| 模块 | 描述 |
|------|------|
| **LCD4002 显示** | 通过软件 I²C 驱动 40×2 字符液晶，实时显示功耗与测量数据 |
| **INA226 功耗监测** | 采集电流（含 1.11× 校正）、母线电压、功率及最大功率 |
| **4×4 矩阵键盘** | 接收用户输入，切换工作模式（0–6）和参数编号（0–9） |
| **UART 指令收发** | 向外部传感器/控制器发送模式指令，接收距离与长度数据 |
| **线性 CCD** | 由 TIM4 定时中断触发 CCD 采集 |
| **电机驱动** | 预留 PWM 电机控制引脚 |
| **编码器** | 支持两路增量式编码器 |
| **IMU 传感器** | 支持 ICM20602 / IMU660RA / IMU963RA / MPU6050 |
| **显示扩展** | 支持 1.8" TFT、1.14" IPS SPI 屏、OLED 屏 |
| **无线通信** | 支持 SI24R1、CH9141 蓝牙、CH573 无线模块（通过 UART4） |
| **电源管理** | 支持 INA219 / INA226 双路功耗监测芯片 |

## 硬件连接

### LCD4002（I²C）

| LCD4002 引脚 | STC8H8K64 引脚 | 说明 |
|---|---|---|
| VCC | 5 V | 电源（或 3.3 V，视模块而定） |
| GND | GND | 地 |
| SDA | P1.4 | 软件 I²C 数据线 |
| SCL | P1.5 | 软件 I²C 时钟线 |

> I²C 默认地址：**0x27**。如地址不同，请修改 `SEEKFREE_LCD4002_I2C.h` 中的 `LCD4002_I2C_ADDR`。

### INA226（I²C）

通过板载 I²C 总线连接，用于采集系统电流、电压与功率。

### 4×4 矩阵键盘

| 信号 | 引脚 |
|---|---|
| 行/列扫描 | P2.0 – P2.7 |

### 调试 UART

| 信号 | 说明 |
|---|---|
| UART1 TX/RX | 默认调试串口，收发测量数据及控制指令 |

## 系统框架

```
用户按键（P2 键盘）
        │
        ▼
   模式/编号选择
        │
        ▼
  UART 发送指令 ──────────────────────────► 外部传感器/控制器
  [0x00][0xFF][Mode][Num][0xFE]
        │                                          │
        │                  ◄──────────────────────┘
        │             UART 接收距离/长度数据
        ▼
  isr.c 协议解析状态机
  → mydistance, mysquarelength
        │
        ▼
  INA226 采集 ──► 电流/电压/功率/最大功率
        │
        ▼
  LCD4002 显示（40×2）
  第0行：MODE / NUMB / 距离 / 长度
  第1行：电流 / 电压 / 功率 / 最大功率
```

## 通信协议

### 发送帧（主控 → 外部设备）

| 字节 | 值 | 说明 |
|---|---|---|
| Byte 0 | `0x00` | 帧头1 |
| Byte 1 | `0xFF` | 帧头2 |
| Byte 2 | `Mode` | 当前工作模式（0–6） |
| Byte 3 | `Num` | 参数编号（0–9） |
| Byte 4 | `0xFE` | 帧尾 |

### 接收帧（外部设备 → 主控）

| 字节 | 值 | 说明 |
|---|---|---|
| Byte 0 | `0x00` | 帧头1 |
| Byte 1 | `0xFF` | 帧头2 |
| Byte 2 | `DistH` | 距离高字节（mm） |
| Byte 3 | `DistL` | 距离低字节（mm） |
| Byte 4 | `LenH` | 长度高字节（mm） |
| Byte 5 | `LenL` | 长度低字节（mm） |
| Byte 6 | `0xFE` | 帧尾 |

接收后自动更新全局变量 `mydistance`（距离，mm）和 `mysquarelength`（长度，mm），并将 `mymode` 重置为 0（空闲）。

## 操作说明

### 键盘布局

```
[ 1 ][ 2 ][ 3 ][ A ]
[ 4 ][ 5 ][ 6 ][ B ]
[ 7 ][ 8 ][ 9 ][ C ]
[ * ][ 0 ][ # ][ D ]
```

### 操作流程

1. **第一次按键**：输入参数编号（0–9），存入缓冲区
2. **第二次按键**：
   - 按数字键（0–9）：确认并设置 `mynubsel`（编号）
   - 按功能键触发模式切换：

| 按键 | 模式 | 说明 |
|---|---|---|
| `A` | Mode 1 | 功能模式 1 |
| `B` | Mode 2 | 功能模式 2（返回时显示最小长度 `Lmi`） |
| `C` | Mode 3 | 功能模式 3 |
| `D` | Mode 4 | 功能模式 4 |
| `#` | Mode 5 | 功能模式 5（使用缓冲编号 `mynubselbuf`） |
| `*` | Mode 6 | 功能模式 6（返回时显示最大长度 `Lma`） |

3. 进入非零模式后，主控通过 UART 持续发送指令，等待外部设备返回数据后自动切回 Mode 0 并刷新显示。

### LCD 显示说明

```
第0行: MODE:x NUMB:x D:xxmm L:xxmm
第1行: Ix.xxxA Vx.xxxV Px.xxxW MAXPx.xxxW
```

- `MODE`：当前工作模式
- `NUMB`：当前参数编号
- `D`：距离测量值（mm）
- `L` / `Lmi` / `Lma`：长度测量值（mm），根据上一次模式显示对应标签
- `I`：电流（A）
- `V`：母线电压（V）
- `P`：实时功率（W）
- `MAXP`：历史最大功率（W）

## 编译与烧录

1. 使用 Keil MDK FOR C51 打开 `Project/MDK/SEEKFREE.uvproj`
2. 点击 **Build** 编译工程，生成 HEX 文件（位于 `Project/MDK/Out_File/`）
3. 打开 **STC-ISP**，选择芯片型号 `STC8H8K64S4`
4. 设置下载串口和波特率，载入 HEX 文件
5. 在 STC-ISP 中将 **P5.4 引脚设置为复位引脚**
6. 点击下载，上电完成烧录

> 系统频率由 `Libraries/libraries/board.h` 中的 `FOSC` 宏控制：设置为 `0` 时自动使用 44.2368 MHz 内部 IRC。
